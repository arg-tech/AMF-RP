Argument Mining Framework for Relation Prediction in Argument pairs.

Expected Input: xAIF file containing the proposition nodes (information nodes) in argument.

Functionality: Predict relations between pairs of propositions (I-nodes), and predicted relations are RA(Default Inference) and CA(Default Conflict)

Expected Output: xAIF file containing the input and new nodes (i.e RA,CA) which are related relations between nodes, and edges information which are connected between I nodes throughput the relation nodes
